# Documentation Directory

This directory contains all documentation for the Quantum Terminal Dashboard.

## Structure

### `guides/`
Comprehensive guides for specific features:
- `*-GUIDE.md` - Feature-specific documentation
- `BUN-TYPESCRIPT-STRIPPER.md` - TypeScript stripping guide
- `YAML-CONFIGURATION-GUIDE.md` - YAML configuration guide
- `FEATURE-FLAGS-GUIDE.md` - Feature flag usage

### Root Documentation
- `README.md` - Main project README
- `QUANTUM-SYSTEM-POLISHED.md` - System overview
- `ENHANCED-PERFORMANCE-ANALYSIS.md` - Performance analysis

## Topics Covered

- **Bun Runtime**: Features, APIs, and best practices
- **TypeScript**: Integration, stripping, and compilation
- **YAML Configuration**: Schema, validation, and usage
- **Feature Flags**: Implementation and management
- **Performance**: Optimization techniques and benchmarks
- **Deployment**: Build and deployment processes
